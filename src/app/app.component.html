Angular 10 Assignment 3

1. Create three new components: GameControl, Odd, and Even

2. The GameControl component should have buttons to start and pause the game

3. When starting the game, an event (holding an incrementing number) should be emitted each second

4. The event should be listenable from outside the component

5. When pausing the game no events should be emitted

6. New Odd and Even components should be generated for each emitted odd and even event

7. Output 'Odd - (number)' or 'Even - (number)' in the two components

8. Finally, style the odd and even components differently to make them distinguishable
